Detalles del Sistema de Navegación y Navbar

Elevando la experiencia de usuario con micro-interacciones cinematográficas

A

Floating Glassmorphism

Una barra flotante con desenfoque extremo (backdrop-filter) y bordes de cristal fino. Al hacer scroll, se encoge o se vuelve más transparente para no distraer.

B

Sidebar Minimalista

Un menú lateral oculto que aparece con una transición elegante, dejando todo el ancho de pantalla para tus renders 3D.

C

Top Integrated (Borders)

Integrada en la parte superior pero sin fondo sólido, usando solo líneas finas rojas para separar secciones, manteniendo el look técnico y limpio.

Heurística: Reducción de Carga Cognitiva

Para la "Immersive Dark Grid", la navegación debe ser casi invisible pero accesible. Usaremos tipografía Oswald en pesos ligeros (200-300) para los enlaces.
